# Prasun Rimal - Portfolio

Personal portfolio for [Prasun Rimal](https://www.linkedin.com/in/prasunrimal/), a software engineer working across full-stack development, backend systems, AI workflows, data analysis, and optimization.

## Live Site

[prasun-rimal.github.io](https://prasun-rimal.github.io)

## Featured Work

- [PayrollOps AI](https://payrollops-ai.vercel.app/login) - AI-assisted payroll exception operations with FastAPI, Next.js, Gemini, LangGraph, PostgreSQL, and pgvector.
- [MockMarket](https://mockmarket-swart.vercel.app) - Full-stack paper-trading platform using Java 21, Spring Boot, React, PostgreSQL, and Finnhub.
- [Manhattan Last-Mile Delivery Optimizer](https://github.com/prasun-rimal/manhattan-delivery-optimizer) - Faculty-advised graph optimization research using Python, Flask, NetworkX, and NYC transportation data.
- [NYC Motor Vehicle Collision Analysis](https://github.com/prasun-rimal/nyc-motor-vehicle-crash-analysis) - Statistical and spatial analysis of more than 187,000 collision records in R.

## Built With

- Semantic HTML
- Responsive CSS
- Lightweight vanilla JavaScript
- GitHub Pages

The site intentionally avoids a frontend framework so it remains fast, accessible, and easy to maintain.

## Run Locally

```bash
git clone https://github.com/prasun-rimal/prasun-rimal.github.io.git
cd prasun-rimal.github.io
python3 -m http.server 8000
```

Open [http://localhost:8000](http://localhost:8000).
